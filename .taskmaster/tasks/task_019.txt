Code Example Snippet (Python):
```python
import requests

def fetch_school_holidays(state_code: str, year: int):
    url = f'https://ferien-api.de/api/v1/holidays/{state_code}/{year}'
    response = requests.get(url)
    response.raise_for_status()
    return response.json()
```

Use this data to update the database accordingly.
