# Product Requirements Document: Robust AI Assistant (Single-User)

## Project Overview

**Project Name:** Robust AI Assistant (Single-User)
**Target User:** Single user (Chefarzt) managing team absence planning
**Architecture:** FastAPI backend, React frontend, SQLite database
**Deployment:** Docker containerized application

## Technical Stack

### Backend
- **Language:** Python
- **Framework:** FastAPI
- **ORM:** SQLModel
- **Database:** SQLite

### Frontend
- **Language:** TypeScript
- **Framework:** React

## Core Features

### 1. Authentication System
- Single hardcoded user authentication
- Secure bcrypt password hashing
- Session management with JWT tokens
- No user registration or multi-user support

### 2. Employee Management
- Create, read, update, delete employee records
- Employee data: name, position/department (optional), annual vacation allowance
- Employee overview with remaining vacation days
- Vacation allowance tracking and calculations

### 3. Absence Management
- Create absence entries for team members
- Absence types: vacation, sick leave, training, etc.
- Date range selection (start date, end date)
- Optional comments for absence entries
- Edit and delete existing absence entries
- Conflict detection for overlapping absences
- All absences are automatically approved (entered by manager)

### 4. Team Calendar View
- Gantt-chart style calendar display
- Horizontal timeline (days/weeks), vertical employee list
- Color-coded absence bars by type
- Monthly and weekly view options
- Scroll and zoom functionality
- Current day highlighting
- Weekend and holiday visual indicators
- Hover/click details for absence entries

### 5. Dashboard Overview
- Key metrics tiles (today's absences, upcoming vacations)
- Remaining vacation days summary per employee
- Quick action buttons for common tasks
- Year-to-date vacation statistics

### 6. Reports and Analytics
- Statistical reports (yearly overview per employee)
- Absence distribution by type
- Monthly absence day counts
- Visual charts (bar charts, pie charts)
- PDF export functionality
- Interactive filters by date range and absence type

### 7. Settings Management
- Database backup functionality
- Database reset functionality
- Absence type management (add, edit, color assignment)
- Holiday calendar configuration
- Password change option

### 8. AI Assistant Integration
- Integration with taskmaster-ai MCP server
- Context7 integration for enhanced AI capabilities
- Perplexity research integration with caching
- AI-powered insights and recommendations

## Data Model

### User Table
- user_id (Primary Key)
- username (Login name)
- password_hash (bcrypt encrypted)
- name (Display name)

### Employee Table
- employee_id (Primary Key)
- name (Full name)
- position (Role/department, optional)
- vacation_allowance (Annual vacation days)
- date_hired (Optional)
- active (Boolean flag)

### Absence Table
- absence_id (Primary Key)
- employee_id (Foreign Key)
- type (Absence type)
- start_date (Start date)
- end_date (End date)
- comment (Optional comment)
- days_count (Calculated field)
- created_at (Timestamp)
- updated_at (Timestamp)

### AbsenceType Table
- type_id (Primary Key)
- name (Type name)
- color_code (Display color)
- counts_as_vacation (Boolean)

### Holiday Table
- date (Date)
- name (Holiday name)
